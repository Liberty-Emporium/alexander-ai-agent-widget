# Alexander AI Agent Widget

Embeddable AI chat widget SaaS. One script tag. Any website. Fully branded.

## Features
- Create AI agents with custom name, color, avatar, system prompt
- One-line embed code — `<script src="...widget/AGENT_ID.js"></script>`
- Powered by OpenRouter (customer brings own API key)
- Chat history, rate limiting, CORS origin control
- Analytics dashboard
- Free / Pro / Business tiers

## Deploy to Railway
1. Fork this repo
2. Create new Railway project → Deploy from GitHub
3. Add volume at `/data`
4. Set env: `SECRET_KEY=<random>`
